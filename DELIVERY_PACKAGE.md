## 🚀 HOW TO GET STARTED

### Step 1: Read Documentation (Choose One)
```
Option A (Fast): Read GETTING_STARTED.sh (5 min)
Option B (Complete): Read INSTALL.md (20 min)
Option C (Comprehensive): Read README.md (15 min)
```

### Step 2: Setup Environment (10 min)
```bash
cp .env.example .env
# Update .env with your values
```

### Step 3: Install Dependencies (15 min)
```bash
cd backend && npm install && cd ..
cd frontend && npm install && cd ..
```

### Step 4: Setup Database (10 min)
```bash
cd backend && npx prisma migrate dev && cd ..
```

### Step 5: Start Development (Ongoing)
```bash
# Terminal 1
cd backend && npm run dev

# Terminal 2
cd frontend && npm run dev

# Open http://localhost:3000
```
